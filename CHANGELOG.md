# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2025-10-12

### Added
- Modern Material Design 3 UI with glassmorphism effects
- Multi-language support (English, Turkish, Russian, Hindi, Chinese)
- Light and Dark theme support
- Multiple download formats (Video+Audio, MP3, Original Audio, Video Only)
- Automatic cookie support for authenticated platforms
- Cross-platform support (Windows, macOS, Linux)
- Embedded binaries (yt-dlp, FFmpeg, FFprobe)
- Download history tracking
- Settings screen with binary status
- Smooth animations and transitions

### Changed
- Complete UI redesign with modern aesthetics
- Improved download progress tracking
- Enhanced error handling
- Better user feedback system

### Technical
- Built with Flutter 3.9.0+
- Uses yt-dlp for video downloads
- FFmpeg for media processing
- Provider for state management
- Google Fonts for typography

## [1.0.0] - Initial Release

- Basic video download functionality
- Simple UI
- Windows support only
